# WorldCupOps Arena 项目方案

## 项目定位

WorldCupOps Arena 是一个以 2026 美加墨世界杯真实赛事数据为基础的 Vibe Coding 教学系统。系统将完整赛制、比赛竞猜、数据可视化、阵容/球员统计和真实数据工程任务结合在同一个本地应用中。

## 系统组成

1. **官方数据同步层**
   - 同步 104 场比赛和 12 组积分榜
   - 默认每小时自动刷新
   - 支持手动刷新和缓存降级

2. **完整赛事浏览**
   - 按阶段、小组、球队和状态筛选
   - 展示小组赛、32 强、16 强、八强、半决赛和决赛

3. **晋级数据可视化**
   - 12 组积分榜
   - 小组积分柱状图
   - 12 支第三名球队跨组排名

4. **单场比赛中心**
   - 官方比分和场地
   - 阵容与比赛事件
   - 球队关键数据
   - 52 人名单及球员统计

5. **竞猜系统**
   - 用户预测与官方结果分离
   - 已完赛场次自动判断准确比分或胜平负

6. **Vibe Coding 教学任务**
   - 任务卡片
   - GLM Coding Plan
   - opencode Execute
   - 验证清单
   - Debug Prompt

## 刷新与可靠性

- 赛事数据 TTL：1 小时
- 进行中比赛详情 TTL：5 分钟
- 已完赛详情 TTL：24 小时
- 官方请求失败：返回最后成功缓存
- 缓存写入：临时文件写入成功后原子替换
- 球员统计失败：继续展示官方阵容

## 现场演示任务

推荐任务：**实现 FIFA 官方数据同步与缓存降级**。

学生需要：

1. 将官方原始 JSON 标准化为稳定结构。
2. 验证同步结果包含 104 场、12 组、48 队。
3. 实现 1 小时 TTL。
4. 模拟官网超时并验证旧缓存仍可使用。
5. 在页面显示数据来源和最后同步时间。

该任务覆盖接口阅读、数据建模、异步请求、缓存、错误处理、测试和 UI 状态反馈，适合深入应用 Vibe Coding。
